Hashish by Neolander

Password generation and management tool

Hashish is a program designed to reduce the hassle of using multiple passwords on each online or offline service which you subscribe to. It predictably generates service passwords from service names and a single master password, and is also able to store the service passwords which you cannot change in an encrypted fashion.

Install

Just unpack the provided binary packages in a place where they won't bother you. For an optimal user experience, I also recommend that you set up an easily accessible shortcut (keystroke, quicklaunch button...) in order to quickly launch Hashish whenever you need a password.
